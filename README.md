# jobmatch_service

Проект представляет собой сервис для хранения, поиска и анализа вакансий. Система позволяет загружать вакансии из подготовленного набора данных, выполнять поиск и фильтрацию по ключевым параметрам, а также автоматически обрабатывать текст описания вакансии: выделять ключевые навыки и определять профессиональную категорию. Проект решает задачу упрощения поиска и анализа релевантных вакансий для пользователя.
